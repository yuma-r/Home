Block across the gap

  • Details:

This game was created as part of a game jam (link) with the theme: Building Bridges. My idea was to develop a game in Unity where you stack blocks of different shapes so that they form a bridge between two platforms. If you succeed, you move on to the next level. The game contains 6 levels, with increasing complexity.

  • Movement:

Here the player character is a cube that can be moved using the arrow keys or WASD. The blocks are stacked using mouse movements. If the character touches the blocks, they can no longer be moved with the mouse so that the player does not have the possibility to “carry” the character to the goal using a block. There are checkpoints in the form of coins that must be activated before the level can be completed. If you fall into the fog, the level resets.

  • Physics:

The directional velocity of the grabbed object is manipulated so that it moves toward the mouse cursor. Gravity is disabled for grabbed objects so that the rotation can be adjusted manually.
